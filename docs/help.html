Памятка по работе с MagicaPixel

Рисование в окне

Все инструменты взаимодействия с изображением в редакторе называются кистями.
Редактор предусматривает несколько кистей на выбор. Все они доступны в левой части приложения:

Кисть для рисования от руки
Кисть для рисования линий
Кисть для заполнения пространства выбранным цветом
Кисть для выбора цвета

Чтобы начать использовать кисть ( рисовать ), надо зажать левую кнопку мыши. Чтобы закончить использовать кисть, нужно отжать левую кнопку мыши.


Слои

В правой части есть окошко с слоями. В тёмном окне отображаются все доступные слои.
Слои можно создавать/удалять, а также перемещать вверх/вниз соответствующими кнопками.
Чуть ниже кнопок есть поле для задания имени слоя. Для изменения имени нужно щёлкнуть по слою и потом отредактировать имя.
Ещё чуть ниже есть поле Alpha blending. Оно отвечает за режим смешения цветов в результирующем изображении на основе альфа-канала. Доступно всего две опции:

ADD Каналы цвета добавляются в результирующее изображение. Количество добавляемого цвета регулирует альфа канал.
OVER Цвет "накладывается" на результирующее изображение, как полупрозрачное стекло. Прозрачность регулируется альфа-каналом.


Импорт/экспорт/создание/масштабирование изображений

В левом верхнем углу есть меню Файл. В нём есть несколько опции для импорта/экспорта/создания/масштабирования изображений соответственно. При выборе "Масштабировать" появляется окно:

Ширина Регулирует ширину изображения
Высота Регулирует высоту изображения
Сохранять соотношение сторон Если включено, то при вводе одного из полей ширина/высота, оставшееся заполняется автоматически с сохранением соотношения сторон.
Масштабировать содержимое холста Если включено, холст масштабируется до заданного размера.
Сглаженное масштабирование Включает сглаженное масштабирование. Края становятся более гладкими. Не имеет эффекта при выключенном масштабировании содержимого холста.

Действия

Ctrl+Z Отменяет действие
Ctrl+Y Восстанавливает отменённое действие
Ctrl+R Устанавливает положение и размер окна холста в изначальное положение
